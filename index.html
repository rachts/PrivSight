Advanced Desktop Privacy Protection with AI

Welcome to Privacy Protector

A sophisticated desktop application that uses facial recognition and artificial intelligence to protect your privacy automatically. When someone approaches your desk or when you're not looking at your screen, your notifications and sensitive content are automatically hidden.

Getting Started

To run the Privacy Protector application:

# Terminal 1: Start Python backend
cd python && python -m server

# Terminal 2: Start Electron dev server
cd electron && npm run dev

Once running, the application will automatically:

  • Connect to your webcam for face detection
  • Monitor system notifications in real-time
  • Blur sensitive content when needed
  • Track your attention and presence
